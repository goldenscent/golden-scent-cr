# Golden Scent CR - sitio web estático

Sitio web listo para publicar gratis con GitHub Pages.

## Archivos principales
- `index.html`: contenido de la página.
- `styles.css`: diseño visual.
- `assets/`: logos, fotos y códigos QR.

## Publicar en GitHub Pages
1. Crear una cuenta en GitHub.
2. Crear un repositorio público llamado `golden-scent-cr`.
3. Subir todos los archivos de esta carpeta al repositorio.
4. Ir a `Settings` > `Pages`.
5. En `Build and deployment`, seleccionar `Deploy from a branch`.
6. En `Branch`, elegir `main` y carpeta `/root`.
7. Guardar.

La página quedará en una dirección parecida a:
`https://TU-USUARIO.github.io/golden-scent-cr/`
